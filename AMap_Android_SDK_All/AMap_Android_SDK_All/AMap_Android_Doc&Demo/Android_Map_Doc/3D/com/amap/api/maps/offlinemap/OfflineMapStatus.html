com.amap.api.maps.offlinemap

类 OfflineMapStatus

  • java.lang.Object
    • com.amap.api.maps.offlinemap.OfflineMapStatus

  • public class OfflineMapStatus
extends java.lang.Object
    地图下载的相关状态码。

    状态说明
    ERROR(<0): 即需要重新下载进行处理,再次开始下载就会自动删除并下载
    EXCEPTION(>100): 暂时出现了问题，还可以继续
    其他均为正常状态

    从以下版本开始:
    V2.0.1
