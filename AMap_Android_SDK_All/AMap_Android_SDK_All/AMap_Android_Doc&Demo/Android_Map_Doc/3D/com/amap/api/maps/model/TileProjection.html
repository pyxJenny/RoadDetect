      • TileProjection

        public TileProjection(int offsetX,
              int offsetY,
              int minX,
              int maxX,
              int minY,
              int maxY)
        根据给定的参数构造一个 TileProjection 新对象。

        例如：用墨卡托投影方式切图后的第一张图片名称为215811_99326.png，最后一张图片名称为215812_99327.png。

        参数:
        offsetX - 从视图左上角（0,0）点计算传入矩形横向偏移像素，单位像素。
        offsetY - 从视图左上角（0,0）点计算传入矩形纵向偏移像素，单位像素。
        minX - 切块后的矩形最小X，如例中的215811。
        maxX - 切块后的矩形最大X，如例中的215812。
        minY - 切块后的矩形最小Y，如例中的99326。
        maxY - 切块后的矩形最大Y，如例中的99327。
        从以下版本开始:
        V2.3.0
