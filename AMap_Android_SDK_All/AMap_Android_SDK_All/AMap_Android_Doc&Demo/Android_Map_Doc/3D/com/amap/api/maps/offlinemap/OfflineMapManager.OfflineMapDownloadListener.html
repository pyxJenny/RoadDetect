      • onDownload

        void onDownload(int status,
              int completeCode,
              java.lang.String name)
        下载状态回调，在调用downloadByCityName 等下载方法的时候会启动
        参数:
        status - 参照OfflineMapStatus
        completeCode - 下载进度，下载完成之后为解压进度
        name - 当前所下载的城市的名字
        从以下版本开始:
        V2.0.1

      • onCheckUpdate

        void onCheckUpdate(boolean hasNew,
                 java.lang.String name)
        当调用updateOfflineMapCity 等检查更新函数的时候会被调用
        参数:
        hasNew - true表示有更新，说明官方有新版或者本地未下载
        name - 被检测更新的城市的名字
        从以下版本开始:
        V2.0.1

      • onRemove

        void onRemove(boolean success,
            java.lang.String name,
            java.lang.String describe)
        当调用OfflineMapManager.remove(String)方法时，如果有设置监听，会回调此方法
        当删除省份时，该方法会被调用多次，返回省份内城市删除情况。
        参数:
        success - ture：删除成功
        name - 所删除的城市的名字
        describe - 删除描述，如 删除成功 "本地无数据"
        从以下版本开始:
        V2.0.1
