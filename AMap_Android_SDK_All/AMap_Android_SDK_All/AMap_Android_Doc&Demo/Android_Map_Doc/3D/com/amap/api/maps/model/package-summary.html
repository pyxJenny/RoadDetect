程序包 com.amap.api.maps.model

覆盖物包，覆盖物（叠加或覆盖到地图的内容）支持标记、折线、多边形和圆。

请参阅: 说明

  • 接口概要 
    接口 说明
    AMapGestureListener
    地图手势识别的回调，包含单双击、滑动等以及地图操作地图后稳定下来的回调
    TileProvider
    接口类，为类TileOverlay提供栅格图像。
