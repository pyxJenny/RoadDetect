      • onRequestFailed

        void onRequestFailed(int lineID,
                   java.lang.String errorInfo)
        轨迹纠偏失败回调
        参数:
        lineID - 用于标示一条轨迹，支持多轨迹纠偏，如果多条轨迹调起纠偏接口，则lineID需不同
        errorInfo - 轨迹纠偏失败原因

      • onTraceProcessing

        void onTraceProcessing(int lineID,
                     int index,
                     java.util.List<LatLng> segments)
        轨迹纠偏过程回调，一条轨迹分割为多个段，按索引顺序回调其中一段
        参数:
        lineID - 用于标示一条轨迹，支持多轨迹纠偏，如果多条轨迹调起纠偏接口，则lineID需不同
        index - 一条轨迹分割为多个段,标示当前轨迹段索引
        segments - 一条轨迹分割为多个段，segments标示当前轨迹段经过纠偏后经纬度点集合

      • onFinished

        void onFinished(int lineID,
              java.util.List<LatLng> linepoints,
              int distace,
              int watingtime)
        轨迹纠偏成功回调
        参数:
        lineID - 用于标示一条轨迹，支持多轨迹纠偏，如果多条轨迹调起纠偏接口，则lineID需不同
        linepoints - 整条轨迹经过纠偏后点的经纬度集合
        distace - 该轨迹经过纠偏后总距离，单位米
        watingtime - 该轨迹中间停止时间，以GPS速度为参考，单位秒
