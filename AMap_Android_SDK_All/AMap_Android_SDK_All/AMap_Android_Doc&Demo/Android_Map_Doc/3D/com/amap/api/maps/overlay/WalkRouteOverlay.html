类 WalkRouteOverlay

  • java.lang.Object
    • com.amap.api.maps.overlay.WalkRouteOverlay

  • public class WalkRouteOverlay
extends java.lang.Object
    步行路线图层类。在高德地图API里，如果要显示步行路线规划，可以用此类来创建步行路线图层。如不满足需求，也可以自己创建自定义的步行路线图层。
    从以下版本开始:
    V2.1.0

                com.amap.api.services.route.WalkPath path,

                com.amap.api.services.core.LatLonPoint end)
        通过此构造函数创建步行路线图层。
        参数:
        context - 当前activity。
        amap - 地图对象。
        path - 步行路线规划的一个方案。详见搜索服务模块的路径查询包（com.amap.api.services.route）中的类 WalkStep。
        start - 起点。详见搜索服务模块的核心基础包（com.amap.api.services.core）中的类LatLonPoint。
        end - 终点。详见搜索服务模块的核心基础包（com.amap.api.services.core）中的类LatLonPoint。
        从以下版本开始:
        V2.1.0
