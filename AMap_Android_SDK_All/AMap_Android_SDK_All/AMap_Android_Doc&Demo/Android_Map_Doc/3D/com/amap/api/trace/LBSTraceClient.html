                       int type,

        参数:
        lineID - 用于标示一条轨迹，支持多轨迹纠偏，如果多条轨迹调起纠偏接口，则lineID需不同
        locations - 一条轨迹的点集合，目前支持该点集合为一条行车GPS高精度定位轨迹

        listener - 轨迹纠偏回调
