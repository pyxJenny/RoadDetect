程序包 com.amap.api.maps.model
的使用

  • 使用com.amap.api.maps.model的程序包 
    程序包 说明
    com.amap.api.maps
    地图显示包，帮助您在Android应用程序中显示高德地图，并且支持地图交互手势的功能。
    com.amap.api.maps.model
    覆盖物包，覆盖物（叠加或覆盖到地图的内容）支持标记、折线、多边形和圆。
    com.amap.api.maps.model.animation  
    com.amap.api.maps.overlay
    图层显示包，包含了对各种交通线路图层以及兴趣点（POI）图层的显示功能。
    com.amap.api.trace
    轨迹纠偏包，提供高精度定位轨迹抓路后绘制平滑轨迹。
  • com.amap.api.maps使用的com.amap.api.maps.model中的类 
    类和说明
    AMapGestureListener
    地图手势识别的回调，包含单双击、滑动等以及地图操作地图后稳定下来的回调
    Arc
    定义了在地图上绘制弧形的类。
    ArcOptions
    圆形选项类
    CameraPosition
    相机位置，这个类包含了所有的可视区域的位置参数。

    定义了在地图上绘制圆形的类。
    CircleOptions
    圆形选项类
    GroundOverlay
    图片层是一张图片以合适的大小贴在地图上。
    GroundOverlayOptions
    图片层的选项类。
    IndoorBuildingInfo
    室内地图属性类，包含室内地图的POIID、楼层总数和当前显示楼层等。
    LatLng
    存储经纬度坐标值的类，单位角度。
    LatLngBounds
    代表了经纬度划分的一个矩形区域。
    Marker
    Marker 是在地图上的一个点绘制图标。
    MarkerOptions
    定义了一个marker 的选项。
    MyLocationStyle
    定位（当前位置）的绘制样式类。
    MyTrafficStyle
    路况拥堵情况对应的颜色
    默认颜色分布为：
    畅通： 0xff00a209
    缓慢： 0xffff7508
    拥堵： 0xffea0312
    严重拥堵： 0xff92000a
    NavigateArrow
    一个导航箭头是多个连贯点的集合，拥有以下属性： 顶点 箭头是由两个顶点之间连贯的点构成的。
    NavigateArrowOptions
    导航箭头的选项类。
    NaviPara
    NaviPara 是唤起高德地图导航功能的参数类。
    Poi
    Poi 是指底图上的一个自带Poi点。
    PoiPara
    唤起高德地图周边搜索功能的参数类。
    Polygon
    这个定义了在地图表面的多边形。
    PolygonOptions
    多边形选项类
    Polyline
    线段类。
    PolylineOptions
    线段的选项类
    RoutePara
    唤起高德地图路径规划功能的参数类。
    Text
    Text 自定义在地图中绘制的文字覆盖物。
    TextOptions
    TextOptions 创建文字覆盖物选项。
    TileOverlay
    TileOverlay是栅格图层的相关类。
    TileOverlayOptions
    TileOverlay的初始化选项配置类。
    TileProjection
    包含矩形按照墨卡托投影切块后的信息类。
    VisibleRegion
    在地图的可视区域中定义多边形四边的四个点坐标包含的区域是可见的。
