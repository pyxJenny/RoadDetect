com.amap.api.maps

类 MapsInitializer

  • java.lang.Object
    • com.amap.api.maps.MapsInitializer

  • public final class MapsInitializer
extends java.lang.Object
    在获取地图之前某些功能想要使用，可以使用这个类初始化高德地图Android API。 由于一些类比如BitmapDescriptorFactory、CameraUpdateFactory需要被初始化，就必须调用类MapsInitializer。 如果您正在使用MapFragment或MapView，并且调用getMap()获得非空的地图，您不需要关注这个类。
    从以下版本开始:
    V2.0.2
